# DV\_RoadTraffic

Road traffic system for Derail Valley.

by GrandpaPlayingGames


## Features

* AI road vehicles
* Route-based traffic system
* Barrier detection (level crossings)
* Editor tools for creating traffic routes



## Installation (NexusMods)

Download from NexusMods and follow instructions there.

https://www.nexusmods.com/derailvalley/mods/1461


## Source Code

This repository contains the full source code of the mod.



## Development Setup



This project is developed as a Derail Valley mod using Unity Mod Manager (UMM).



To build this mod, create a Class Library project and add references to the following assemblies.



### Notes



\- Target framework: .NET Framework 7.3 

\- Platform: x64 (recommended)

\- Do NOT copy this repository as a full project—create your own and add these source files.
